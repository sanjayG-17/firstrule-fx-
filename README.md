# FirstRule fx

A portfolio website for editing to showcase our video editing skills and other skills.

## Project info

This project was built with Lovable and Vite.

## How can I edit this code?

There are several ways of editing your application.

**Use Lovable**

Simply visit your project on [Lovable](https://lovable.dev) and start prompting.

**Use your preferred IDE**

If you want to work locally using your own IDE, you can clone this repo and push changes.

The only requirement is having Node.js & npm installed.

Follow these steps:

```sh
# Step 1: Install dependencies
npm i

# Step 2: Start the development server
npm run dev
```

## What technologies are used?

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
